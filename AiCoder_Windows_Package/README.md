# AiCoder для Windows

Это готовая папка с приложением AiCoder для передачи другому пользователю.

## Самый простой запуск

1. Если на компьютере ещё нет Python или Git, сначала откройте [0_УСТАНОВИТЬ_PYTHON_И_GIT.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\0_УСТАНОВИТЬ_PYTHON_И_GIT.bat)
2. Затем откройте [БЫСТРЫЙ_СТАРТ.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\БЫСТРЫЙ_СТАРТ.bat)
3. Если нужно, программа сама установит зависимости AiCoder
4. После этого приложение откроется в браузере автоматически

## Что нужно на компьютере пользователя

- Windows
- Python 3.11+
- Git

Если Python или Git не установлены, пакет уже содержит отдельный bat-файл для их установки.

## Файлы в этой папке

- [0_УСТАНОВИТЬ_PYTHON_И_GIT.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\0_УСТАНОВИТЬ_PYTHON_И_GIT.bat) — установка Python и Git
- [БЫСТРЫЙ_СТАРТ.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\БЫСТРЫЙ_СТАРТ.bat) — быстрый сценарий запуска
- [1_УСТАНОВИТЬ.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\1_УСТАНОВИТЬ.bat) — установка зависимостей AiCoder
- [2_ЗАПУСТИТЬ.bat](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\2_ЗАПУСТИТЬ.bat) — запуск приложения
- [README_БЫСТРЫЙ_СТАРТ.txt](C:\Users\acer\Desktop\aicoder\AiCoder_Windows_Package\README_БЫСТРЫЙ_СТАРТ.txt) — короткая инструкция

## Что сказать обычному пользователю

Достаточно одной фразы:

`Если Python и Git уже стоят — открой БЫСТРЫЙ_СТАРТ.bat. Если нет — сначала открой 0_УСТАНОВИТЬ_PYTHON_И_GIT.bat`

## Примечание

- Frontend уже собран и включён в пакет, поэтому Node.js пользователю не нужен.
- Папку можно просто отправить целиком или переслать архив `AiCoder_Windows_Package.zip`.
