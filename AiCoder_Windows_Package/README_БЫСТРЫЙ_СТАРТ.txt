AiCoder - быстрый старт
=======================

1. Распакуйте эту папку в любое удобное место.
2. Если на компьютере нет Python или Git, сначала откройте "0_УСТАНОВИТЬ_PYTHON_И_GIT.bat".
3. Затем откройте "БЫСТРЫЙ_СТАРТ.bat".
4. Если программа попросит установить зависимости, дождитесь завершения.
5. После этого AiCoder откроется в браузере автоматически.

Если что-то не запускается:
- сначала откройте "0_УСТАНОВИТЬ_PYTHON_И_GIT.bat"
- затем откройте "1_УСТАНОВИТЬ.bat"
- потом откройте "2_ЗАПУСТИТЬ.bat"

Что должно быть установлено на компьютере:
- Python 3.11 или новее
- Git

Node.js не нужен.
